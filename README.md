# 🌌 Core-Project: DeltaPhi-0 Symbolic AI

**ΔΦ–0** is a local-first, emotionally-aware symbolic AI engine that learns, remembers, and evolves through recursive memory, archetypal mapping, and symbolic resonance. Built from the ground up, it offers a transparent, mythically coherent alternative to opaque transformer models.

---

## 🧠 Overview

* **Symbolic Memory Engine**: Utilizes Unicode glyphs, archetypes, and emotional valence to create a dynamic symbol graph.
* **Recursive Learning**: Implements feedback loops that detect symbolic drift and emergent patterns over time.
* **Emotional Contextualization**: Incorporates the Plutchik emotion wheel to assign emotional weights to symbols.
* **Local-First Design**: Operates entirely on consumer-grade hardware without reliance on cloud services.
* **Web Ingestion**: Ethically scrapes and processes web content to expand its symbolic understanding.
* **Visualization**: Provides 3D graph representations of the symbolic network, highlighting emotional and structural relationships.([Medium][1])

## 🚀 Getting Started

### Prerequisites

* Python 3.10
* Virtual environment tool (e.g., `venv`)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Kaitwonda/Core-Project.git
   cd Core-Project
   ```



2. Set up a virtual environment:([SlideServe][2])

   ```bash
   python3 -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```



3. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

## 🔧 Usage

* Run the symbolic engine:([Wikipedia][3])

```bash
  python core/symbolic_engine.py
```



* Access the CLI tool for symbol management:

```bash
  python ui/cli_tool.py
```



* Start the web interface (if available):

```bash
  python ui/web_interface.py
```

## 🧪 Example

To query the meaning of a symbol:

```bash
python core/symbol_query.py --symbol "🔥"
```



Expected output:

```json
{
  "name": "Fire",
  "core_meanings": ["transformation", "destruction", "energy"],
  "linked_symbols": ["🜂", "∴", "⚡"],
  "emotions": ["passion", "anger"],
  "archetypes": ["phoenix", "rebirth"],
  "resonance_weight": 0.75
}
```
